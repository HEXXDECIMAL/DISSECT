DISSECT

Deep static analysis for threat detection. AST-aware, not regex-blind.

What is DISSECT?

  • AST-Aware: Understands code semantics across 20+ languages to detect intent, not just strings.
  • Deep Binary Analysis: Leverages Radare2/Rizin and Goblin for deep inspection of Mach-O, ELF, and PE.
  • Recursive Inspection: Automatically unpacks archives (ZIP, TAR, 7z, RAR) and inspects nested content.
  • Supply Chain Defense: Built to identify hostile behaviors in packages, extensions, and source code.

Supported Formats

  • Binaries: ELF, Mach-O, PE executables and libraries
  • Source: Go, Rust, Python, JS/TS, C/C++, Java, Ruby, PHP, and more
  • Packages: npm, Chrome extensions, VSCode extensions
  • Archives: ZIP, TAR, 7z, RAR, XAR, JAR
